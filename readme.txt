This Ovulation Calculator will calculate your period, fertility and shows the next 6 months ovulation dates.

== Description ==

This Ovulation Calculator will calculate your most fertility days. You can use shortcode to page or post or sidebar. The form itself and the text results are mobile friendly and responsive, built with usability in mind. The Ovulation Calculator for WordPress Plugin can be successfully used in desktop, mobile and tablet environment.

== Installation ==

1. Upload the plugin directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Ovulation Calculator section in WordPress dashboard and enter details.


== Frequently Asked Questions ==

= Does the calculator have shortcode ? =

Yes. The shortcode is [ovulationcalculator]. You can use it both in post and page and even in sidebar.

= Can I translate/rewrite the wordings of this plugin ? =

In terms of localisation, you are free to change any text and / or date format, from plain English to any language, from the settings page.
